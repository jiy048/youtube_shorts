## 🚀 **Quick Setup Commands**

### **Option 1: Automated Setup**
```bash
# 1. Create project directory
mkdir youtube-shorts-recommendation
cd youtube-shorts-recommendation

# 2. Download and run setup script
curl -O https://raw.githubusercontent.com/your-repo/setup.sh  # You'll need to host this
chmod +x setup.sh
./setup.sh

# 3. Copy all Python files to src/ directory
# 4. Copy configuration files to their locations
# 5. Run the pipeline
make full-pipeline
```

### **Option 2: Manual Setup**
```bash
# 1. Create directory structure
mkdir -p youtube-shorts-recommendation/{src/utils,tests,scripts,monitoring/grafana/dashboards,.github/workflows,config}
cd youtube-shorts-recommendation

# 2. Create setup script
# Copy the setup.sh content and save it as scripts/setup.sh
chmod +x scripts/setup.sh
./scripts/setup.sh

# 3. Copy all artifacts to their respective locations (see structure above)

# 4. Run setup
source venv/bin/activate
python src/data_generator.py
python src/train.py
uvicorn src.api:app --reload
```

## 🔧 **File Import Path Updates**

Since we moved files to `src/` directory, update import paths:

### **In train.py:**
```python
# Change this:
from feature_engineering import FeatureEngineer
from data_validation import validate_processed_data

# To this:
from src.feature_engineering import FeatureEngineer
from src.data_validation import validate_processed_data
```

### **In api.py:**
```python
# Change this:
from feature_engineering import FeatureEngineer

# To this:
from src.feature_engineering import FeatureEngineer
```

### **In test_api.py:**
```python
# Add at the top:
import sys
import os
sys.path.append(os.path.dirname(os.path.dirname(os.path.abspath(__file__))))
```

## 🎯 **Verification Checklist**

After copying all files, verify your structure:

```bash
# Check directory structure
tree -I 'venv|__pycache__|*.pyc|.git' .

# Verify Python files
find . -name "*.py" -type f | head -10

# Check Docker files
ls -la Docker*

# Verify configuration
ls -la monitoring/
ls -la .github/workflows/
```

## 🚀 **Final Execution**

Once everything is in place:

```bash
# 1. Activate environment
source venv/bin/activate

# 2. Generate data
python src/data_generator.py

# 3. Train model  
python src/train.py

# 4. Start API
uvicorn src.api:app --reload --port 8000

# 5. Start UI (in new terminal)
streamlit run src/streamlit_app.py --server.port 8501
```
